My implemented project has been fully tested on niosII_9.0sp2. Follow the steps below to check it out
1.	unpack the project source code
2.	create a blank "C" (C++ is not supported) project called "blank_project_0"
3.	drag all of the 8 ".h", ".c" and ".S" files into "blank_project_0" folder of the "Project Navigator Window" of your NIOS-II IDE
4.	compile the project and run it on DE-2
